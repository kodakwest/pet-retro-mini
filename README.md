# PET Retro Mini

A compact (~3-5 MB) standalone Windows .exe Commodore PET emulator. Like a retro mini console — for your desktop.

5 iconic games ready to play. Import any .prg file for thousands more.

## Quick Start

1. Download the latest release from GitHub Releases
2. Unzip anywhere
3. Run `pet-retro-mini.exe`
4. Pick a game and play

## MVP Games

- **Zork I** — The great underground adventure. MIT licensed.
- **Miner** — Navigate a mine shaft collecting minerals. Public domain.
- **Lunar Lander** — Pilot the landing module. Public domain.
- **Dungeon** — Early dungeon crawl RPG. Public domain.
- **PET Invaders** — Classic space shooter. Freeware.

## Import Games

Drag any .prg file onto the window. It loads and runs. That's it.
